# v0.1.1 — Companion browsers (morphology, manuscript timeline)

**Release date:** 2026-05-04
**Status:** Private academic prototype — not for public distribution. (Unchanged from v0.1.0.)

---

## What is added

- **Manuscript timeline browser** — `src/components/ManuscriptTimeline.jsx`. Plots C-14 ranges of four early Qur'an manuscripts (Parisino-petropolitanus, Birmingham, Sanaa, Topkapi) on a shared axis with a detail panel for the selected codex. Author-written summaries; image URLs reference Wikimedia Commons.
- **Morphology browser** — `src/components/MorphologyBrowser.jsx`. Side-by-side comparison of **QAC** (Quranic Arabic Corpus, GPL-3.0) and **Talmon** root/lemma/POS analyses, highlighting disagreements. Currently a small sample (Q 1:4, Q 2:9).
- **View switcher in `App.jsx`** — header now toggles between Variants / Morphology / Manuscripts views.
- **Three new build scripts:**
  - `scripts/build-hafs-corpus.js` — copies upstream KFGQPC Hafs into `src/data/hafsCorpusData.json` for full-corpus browsing. Strips trailing verse-number markers only; never generates text. Output is git-ignored.
  - `scripts/fetch-morphology.js` — writes `src/data/morphologyData.json` from a hardcoded QAC/Talmon sample plus surface forms pulled by index from `quranData.json`.
  - `scripts/extract-manuscripts.js` — regex-extracts dates, dimensions, and image URLs from inlined manuscript prose into `src/data/manuscriptsData.json`.
- **Tests** — `tests/components.test.jsx` extended with `<MorphologyBrowser>` and `<ManuscriptTimeline>` cases.

## What is not added

- No new readings, no new verses, no new variant groups in `quranData.json`.
- No promotion of any group from `curated_sample` to `verified`.
- No change to `verificationLedger.json` semantics or `_meta.generationPolicy`.
- No bundled raw corpora.

## Sacred text integrity reaffirmed

- `_meta.generated: false` and `_meta.generationPolicy: "no-arabic-text-generation"` unchanged.
- `MorphologyBrowser` displays Arabic surface forms by reading `quranData.json` tokens at the indicated index; it never synthesises text.
- `ManuscriptTimeline` displays no Qur'anic text; only metadata, dates, and dimensions.
- `build-hafs-corpus.js` performs pure copy + verse-number-marker stripping (regex `/[\s ]*[٠-٩۰-۹]+[\s ]*$/u` on each ayah). Output is treated as untrusted upstream redistribution and is git-ignored.

## Data licensing notes

- **`src/data/hafsCorpusData.json`** — `.gitignore`d. Built from `scripts/_raw/kfqpc-hafs.json` whose `authorityStatus` is `unknown` (`src/data/sourceManifest.json`). KFGQPC redistribution terms are unconfirmed; treat the output as a local-only browsing aid.
- **`src/data/morphologyData.json`** — committed. Uses GPL-3.0 QAC root/lemma/POS values for ~4 word slots; QAC is now listed in the README data-licensing table.
- **`src/data/manuscriptsData.json`** — committed. Author-written prose; image URLs link to Wikimedia Commons (per-image licenses, no bundling).
